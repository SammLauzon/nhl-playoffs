# Classement
Voici comment les points sont calculés pour cette année:

* 2 pts par victoire pour l'équipe choisie.
* +2 pts si l'équipe choisie est victorieuse.
* +3 pts si l'équipe choisie est victorieuse et que le nombre de matchs choisi est exact.
* Ceci donne donc un maximum de 13 points par série.
